# Markdown Konverzija (Pandoc)

---

## Primjeri konverzije

### 1) .md → .html

```
pandoc dokument.md -o dokument.html
```

Pretvara Markdown datoteku u HTML.

### 2) .md → .pdf

```
pandoc dokument.md -o dokument.pdf
```

Pretvara Markdown datoteku u PDF.

### 3) .md → .docx

```
pandoc dokument.md -o dokument.docx
```

Pretvara Markdown datoteku u Word dokument.

### 4) .md → .txt

```
pandoc dokument.md -o dokument.txt
```

Pretvara Markdown u običan tekst.
